# Part 3 Repository Content

This folder contains the application manifests used in Part 3.

## Instructions
1. Create a public GitHub repository named: `42_IoT_part3_nmunir`.
2. Copy the `app` folder from this directory to the root of your new repository.
3. Commit and push the changes:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of application manifests"
   git branch -M main
   git remote add origin https://github.com/nmunir/42_IoT_part3_nmunir.git
   git push -u origin main
   ```
4. Verify that Argo CD in your K3d cluster syncs the application.
